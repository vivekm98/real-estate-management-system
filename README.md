# 🏠 Real Estate Management System

A **Full Stack Real Estate Web Application** where admins can manage property listings and users can explore properties by category, view property details, and contact owners.

This project is built using **React.js for the frontend** and **ASP.NET Core Web API with Entity Framework Core for the backend**.

---

## 🚀 Features

### Admin Features

* Add new property categories
* Upload category images
* Add properties with details and images
* Edit and update property information
* Delete properties or categories
* Mark properties as **Trending**

### User Features

* View property categories
* Filter properties by category
* View property listings
* See **Trending Properties**
* View full property details
* Get owner contact information

## ⚙️ Installation

### 1️⃣ Clone the Repository

```
git clone https://github.com/yourusername/real-estate-management-system.git
```

---

### 2️⃣ Backend Setup (ASP.NET Core)

Open backend project and run:

```
dotnet restore
dotnet run
```

API will run on:

```
https://localhost:7225
```

---

### 3️⃣ Frontend Setup (React)

Go to frontend folder:

```
cd frontend
npm install
npm run dev
```

Frontend will run on:

```
http://localhost:5173
```
